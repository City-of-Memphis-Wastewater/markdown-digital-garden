Description

This course is designed for personnel whose primary interest and/or job assignment includes maintaining the Ovation hardware. Select topics from several courses are incorporated and expanded upon, including the replacement and set up of an Ovation controller & flash disk, and installing new I/O thermocouple/RTD modules and building several thermocouples and RTD points. Several power supply configurations will be discussed. Students will install a HART 4-20 mA analog input I/O module and transmitter as well as install and configure an Ovation remote node controller with fiber connections. Hands-on exercises will require building control sheets and linking digital input and relay output modules.

Objectives

1. Identify the typical Ovation Distributed Control System Components
2. Identify the components that make up an OCR400/OCR100 Controller
3. Understand the Ovation Network & I/O Addressing for Local and Remote I/O
4. Examine 2 different power supply configuration
5. Examine the System Viewer Application
6. Build Digital Points for an existing module
7. Describe Module Light Information
8. Insert and Apply a Thermocouple module
9. Insert and Apply an RTD Module
10. Insert and Apply an Analog Input 4-20 mA module
11. Perform Coefficient Calculations on Analog Input Points
12. Introduction to Ovation Control Builder
13. Discuss and create Node Points
14. Install a Remote I/O Node
15. Insert a Relay Output Module in a Remote Node
16. Insert a Contact Input Module in a Remote Node
